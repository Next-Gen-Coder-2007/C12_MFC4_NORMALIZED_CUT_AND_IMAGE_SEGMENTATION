# Image Segmentation Web Application

A web-based application built using **Python** and **Flask** that allows users to upload images and apply multiple classical **image segmentation techniques**. This project is intended for learning, experimentation, and comparison of segmentation algorithms.

---

## 📌 Overview

This application provides an interactive interface for exploring image segmentation methods commonly used in computer vision. Users can upload an image, select a segmentation technique, and instantly view the processed result.

---

## ✨ Features

- Upload images through a web interface  
- Apply multiple image segmentation algorithms  
- Automatically save segmentation results  
- Maintain a history of processed images  
- Simple and user-friendly interface  

---

## 🧠 Segmentation Techniques

The following segmentation methods are implemented:

- **K-Means Clustering**
- **Normalized Cut**
- **Thresholding**
- **Edge Detection**
- **Region Growing**

Each method segments the image differently based on pixel similarity, intensity, or boundaries.
