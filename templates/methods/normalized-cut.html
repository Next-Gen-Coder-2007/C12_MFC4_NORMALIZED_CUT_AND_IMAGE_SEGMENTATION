Normalized Cut Segmentation

1. Problem Definition

Image segmentation aims to partition an image into meaningful and homogeneous regions. Normalized Cut models this problem using graph theory, where segmentation is formulated as a graph partitioning task.

2. Graph Representation

An image is represented as an undirected weighted graph:

$$G = (V, E)$$
  • V — nodes representing pixels or superpixels
  • E — edges between neighboring nodes
  • W(i,j) — similarity between nodes

3. Similarity Function

The similarity between two nodes is computed using color and spatial proximity:

$$W(i,j) = \exp\left(-\frac{||F_i - F_j||^2}{\sigma_I^2}\right) \cdot \exp\left(-\frac{||X_i - X_j||^2}{\sigma_X^2} \right)$$

High weights indicate strong similarity, while small weights indicate weak relationships.

4. Minimum Cut Limitation

A naive partition minimizes the cut value:

$$cut(A,B) = \sum_{i \in A, j \in B} W(i,j)$$

However, minimum cut tends to isolate small regions, producing trivial segmentations.

5. Normalized Cut Criterion

To overcome this bias, the normalized cut objective is defined as:

$$NCut(A,B) = \frac{cut(A,B)}{assoc(A,V)} + \frac{cut(A,B)}{assoc(B,V)}$$

This normalization enforces balanced partitioning of the graph.

6. Matrix Formulation

$$D(i,i) = \sum_j W(i,j)$$ $$L = D - W$$

The optimization problem becomes:

$$\min_x \frac{x^T (D - W)x}{x^T D x}$$

7. Eigenvalue Solution

Solving this leads to the generalized eigenvalue problem:

$$(D - W)x = \lambda D x$$

The second smallest eigenvector (Fiedler vector) provides the optimal bipartition.

8. Segmentation Decision

$$Label(i) = \begin{cases} A & x_i > 0 \\ B & x_i \le 0 \end{cases}$$

Each component of the eigenvector determines the region assignment of the node.

Algorithm Workflow

  • Generate superpixels using SLIC
  • Extract mean color and spatial coordinates
  • Construct adjacency graph
  • Compute affinity matrix W
  • Build degree matrix D
  • Solve generalized eigenvalue equation
  • Threshold eigenvector values
  • Map labels back to image pixels

Interpretation

Normalized Cut performs global segmentation by analyzing the entire image structure. Unlike local thresholding methods, it optimizes segmentation at the graph level, producing balanced and meaningful regions.
