Thresholding Segmentation

Mathematical Formulation

1. Binary Thresholding

$$g(x,y)=\begin{cases}1,& f(x,y)>T\\0,& f(x,y)\le T\end{cases}$$

f(x,y) = pixel intensity, g(x,y) = segmented output, T = threshold value.

2. Otsu's Method

Optimal threshold maximizes between-class variance:

$$\sigma_B^2(T) = \omega_0(T)\omega_1(T)[\mu_0(T)-\mu_1(T)]^2$$

ω₀, ω₁ = class probabilities; μ₀, μ₁ = class means.

3. Class Probabilities

$$\omega_0(T)=\sum_{i=0}^{T} p(i),\quad \omega_1(T)=\sum_{i=T+1}^{L-1} p(i)$$

4. Class Means

$$\mu_0(T)=\frac{\sum_{i=0}^{T} i \cdot p(i)}{\omega_0(T)},\quad \mu_1(T)=\frac{\sum_{i=T+1}^{L-1} i \cdot p(i)}{\omega_1(T)}$$

5. Optimal Threshold

$$T^*=\arg\max_T \sigma_B^2(T)$$

6. Multi-Level Thresholding

$$ g(x,y) = \begin{cases} 0, & \text{if } f(x,y) \le T_1 \\ 1, & \text{if } T_1 < f(x,y) \le T_2 \\ \vdots & \\ n, & \text{if } f(x,y) > T_n \end{cases} $$

Workflow

  1. Convert the image to grayscale (if it is colored).
  2. Compute histogram and class probabilities p(i).
  3. Choose threshold manually or use Otsu's method.
  4. Assign pixels using the binary function g(x,y).
  5. Apply multi-level thresholding for multiple classes if required.
  6. Optional: Apply morphological operations for noise removal.
