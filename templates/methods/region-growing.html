Region Growing Segmentation

Overview

Region growing is a bottom-up segmentation technique that starts with seed points and iteratively adds neighboring pixels with similar properties. It groups pixels into regions based on predefined similarity criteria, creating homogeneous segments representing distinct objects or areas.

Mathematical Foundation

1. Similarity Predicate

$$P(R \cup \{p\}) = \text{TRUE}$$

P is a logical predicate (similarity criterion).

2. Homogeneity Criterion

$$|I(p) - \mu_R| < T$$

Where:

  • I(p) = intensity/color of pixel p
  • μR = mean intensity of region R
  • T = threshold for similarity

3. Color Distance in LAB Space

$$\Delta E = \sqrt{(L_p - L_s)^2 + (a_p - a_s)^2 + (b_p - b_s)^2}$$

(L, a, b) are LAB color components; p = pixel, s = seed.

4. Region Mean Update

$$\mu_R^{new} = \frac{n \cdot \mu_R^{old} + I(p)}{n + 1}$$

n = current number of pixels in region R.

5. Connectivity

$$N_4(x,y) = \{(x\pm1, y), (x, y\pm1)\}$$
$$N_8(x,y) = N_4(x,y) \cup \{(x\pm1, y\pm1)\}$$

6. Stopping Criterion

$$\forall p \in \text{boundary}(R): |I(p) - \mu_R| \geq T$$

7. Multi-Seed Growing

$$R_i = \{p : i = \arg\min_j \Delta E(p, seed_j)\}$$

Algorithm Steps

  1. Seed Selection: Choose initial seed points (manual or automatic)
  2. Initialize: Create empty regions and mark seeds
  3. Queue Neighbors: Add neighbors of seed to processing queue
  4. Similarity Check: Test if neighbor meets homogeneity criterion
  5. Add to Region: If similar, add pixel and queue its neighbors
  6. Update Statistics: Recalculate region mean/variance
  7. Repeat: Continue until queue is empty
  8. Post-process: Merge small regions or fill holes
