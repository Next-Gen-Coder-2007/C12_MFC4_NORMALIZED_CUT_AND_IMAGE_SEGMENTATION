K-Means Image Segmentation

Mathematical Representation

Let an image contain N pixels. Each pixel is represented as a feature vector:

$$x_i = [R_i, G_i, B_i]^T$$

The image is transformed into a dataset:

$$X = \{x_1, x_2, ..., x_N\}, \quad x_i \in \mathbb{R}^3$$

Objective Function

K-Means partitions pixels into K clusters by minimizing intra-cluster variance:

$$J = \sum_{k=1}^{K} \sum_{x_i \in C_k} \|x_i - \mu_k\|^2$$

where μk is the centroid of cluster k.

Distance Computation

$$d(x_i,\mu_k) = \sqrt{(R_i-R_k)^2+(G_i-G_k)^2+(B_i-B_k)^2}$$

Assignment Rule

$$C_k = \{x_i : \|x_i-\mu_k\| \le \|x_i-\mu_j\|,\ \forall j\}$$

Centroid Update

$$\mu_k = \frac{1}{|C_k|}\sum_{x_i \in C_k} x_i$$

Iterative Optimization

K-Means alternates between assignment and update steps. Each iteration monotonically reduces the objective function:

$$J^{(t+1)} \le J^{(t)}$$

Convergence Condition

$$\|\mu_k^{(t+1)} - \mu_k^{(t)}\| < \epsilon$$

Final Segmentation

After convergence, each pixel is labeled by its cluster index:

$$\text{label}(x_i) = k$$

The segmented image is obtained by replacing each pixel with its centroid color:

$$x_i^{seg} = \mu_{\text{label}(x_i)}$$

Complete Workflow

  1. Convert image to pixel feature vectors
  2. Initialize K centroids
  3. Compute distance between pixels and centroids
  4. Assign pixels to nearest centroid
  5. Update centroids using mean operation
  6. Repeat until convergence
  7. Form segmented image using cluster labels
